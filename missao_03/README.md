# Missão 3 – O Núcleo da Manipulação Tripla
Está na hora de abandonar os atalhos e moldar o espaço com suas próprias mãos, Guardião(ã).
Em seu laboratório de modelagem, você deverá criar um objeto tridimensional único, sem o auxílio dos
artefatos prontos da GLUT. Mas apenas construí-lo não basta, você deve ser capaz de manipulá-lo com
maestria em todos os três eixos, aplicando transformações geométricas compostas.
Mostre tudo que você é capaz de fazer seguindo os passos abaixo:
• Modele manualmente um objeto 3D (ex: pirâmide, prisma, robô minimalista, estrutura composta de cubos
ou polígonos)
• Aplique duas transformações geométricas (positivas e negativas) para cada eixo (X, Y e Z), por exemplo:
• Translação no eixo X com 't' / 'T'
• Rotação no eixo Y com 'y' / 'Y'
• Escala no eixo Z com 'e' / 'E'
• As transformações devem ser realizadas combinando funções como glTranslatef, glRotatef, glScalef.
