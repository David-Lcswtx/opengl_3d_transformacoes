# Missão 5 – O Encontro das Sombras
Guardião(ã), a travessia do espaço não se completa apenas com força e controle. Toda forma carrega
consigo uma contraface, um reflexo oculto, uma versão invertida que só se revela diante dos Eixos da
Verdade.
Seu desafio é evocar o Reflexo do Objeto, espelhando sua essência em relação aos eixos do mundo visível.
A cada espelhamento, você revelará uma parte oculta da sua criação, um eco gráfico daquilo que já existe e
do que poderia ser. Para encontrar sua própria sombra, você deve:
• Desenhar um objeto 2D qualquer (ex: símbolo tribal, letra grega, relíquia geométrica)
• Mostrar os eixos X e Y na tela como limiares dimensionais
• Aplicar espelhamentos com:
• 'x' → invoca o reflexo no eixo X (como se a figura caísse na água)
• 'y' → invoca o reflexo no eixo Y (como se fosse vista num espelho lateral)
• Dica do Dr. Sombrio: use glScalef com valores negativos para realizar as inversões
